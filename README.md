# みみぞの大冒険

「みみぞの」を主人公にした、パステルカラーの横スクロールアクションゲームです。  
スーパーマリオ風の操作感で、星を集めながらゴールの旗を目指しましょう！

## 遊び方

1. `index.html` をブラウザで開く
2. **スペースキー** または **画面タップ** でスタート
3. 星を集めてスコアアップ（1個10点）
4. 敵の上から踏むと倒せる（20点）
5. 旗に到達するとステージクリア！

## 操作方法

### PC

| 操作 | キー |
|------|------|
| 左へ移動 | ← |
| 右へ移動 | → |
| ジャンプ（二段ジャンプ可） | スペース |
| スタート / リトライ | スペース |

### スマホ

| 操作 | ボタン |
|------|--------|
| 左へ移動 | 画面左の ◀ |
| 右へ移動 | 画面左の ▶ |
| ジャンプ | 画面右の ★ |

## ゲームのルール

- **ライフ**：最初は3つ。敵に横から当たるか、穴に落ちると1つ減る
- **敵**：眠気・洗い物・便秘の3種類。上から踏むと倒せる
- **ゲームオーバー**：ライフが0になると終了。スペースまたはタップでリトライ

## ファイル構成

```
├── index.html    … ゲーム画面
├── style.css     … デザイン
├── script.js     … ゲームロジック
├── player.png    … 主人公「みみぞの」の画像
└── README.md     … このファイル
```

## 起動方法（初心者向け）

### 方法1：ダブルクリック

1. フォルダ内の `index.html` を探す
2. ファイルを **ダブルクリック**
3. ブラウザ（Safari、Chrome など）でゲームが開きます

### 方法2：ブラウザにドラッグ

1. ブラウザを開く
2. `index.html` をブラウザのウィンドウに **ドラッグ＆ドロップ**

### 方法3：ターミナルから開く（Mac）

```bash
open index.html
```

## 注意事項

- インターネット接続は不要です（すべてローカルで動作）
- 外部ライブラリは使っていません
- 効果音・BGMはブラウザが自動生成します（昭和レトロ風のチップチューンBGM）
- 初回プレイ時は、画面をクリックまたはタップして音声を有効にしてください

## ヒント

- ジャンプは空中でもう一度押せる（二段ジャンプ）
- 敵に横から当たるとダメージ。上から踏むのが正解！
- 穴の前でジャンプのタイミングに注意

## GitHub Pages で公開する手順

### 1. GitHub にリポジトリを作成

1. [GitHub](https://github.com) にログイン
2. 右上の **＋** → **New repository**
3. リポジトリ名を入力（例：`mimizono-adventure`）
4. **Public** を選択
5. **Create repository** をクリック

### 2. ファイルをアップロードしてプッシュ

ターミナルで、このフォルダに移動して以下を実行します。

```bash
cd "/Users/anaguchiyuko/Desktop/Threads運用"

git init
git add index.html style.css script.js player.png README.md .gitignore
git commit -m "みみぞの大冒険を公開"

git branch -M main
git remote add origin https://github.com/あなたのユーザー名/mimizono-adventure.git
git push -u origin main
```

`あなたのユーザー名` と `mimizono-adventure` は、作成したリポジトリに合わせて変更してください。

### 3. GitHub Pages を有効化

1. GitHub のリポジトリページを開く
2. **Settings** → 左メニューの **Pages**
3. **Build and deployment** の **Source** で **Deploy from a branch** を選択
4. **Branch** を `main`、フォルダを `/ (root)` に設定
5. **Save** をクリック

数分後、次のURLでゲームが公開されます。

```
https://あなたのユーザー名.github.io/mimizono-adventure/
```

### 4. 更新したとき

ゲームを修正したら、同じフォルダで以下を実行します。

```bash
git add .
git commit -m "ゲームを更新"
git push
```

1〜2分後にサイトへ反映されます。
